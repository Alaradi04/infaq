# INFAQ – AI-Powered Personal Financial Management App

INFAQ is a mobile financial management application designed to help individuals take control of their spending and savings. In today’s fast-paced world, it is easy to lose track of where money goes — whether on food, clothing, subscriptions, or impulse purchases.

INFAQ acts as a personal finance assistant by tracking expenses, analyzing behavioral patterns, and generating intelligent insights that encourage better financial decisions. Whether saving for a vacation, a new device, or simply aiming to reduce unnecessary spending, INFAQ supports users in building healthier financial habits.

> This project is developed as a Senior Project at the University of Bahrain.

---

## 📌 Project Overview

Many individuals struggle to manage daily expenses due to unclear spending patterns, impulsive purchases, and limited financial awareness. INFAQ addresses these challenges by providing:

- Manual expense tracking  
- AI-assisted expense categorization  
- Spending pattern analysis  
- Budget monitoring and intelligent alerts  
- Sustainability-focused spending insights  
- Visual dashboards and summaries  

> The system emphasizes *lightweight AI models* and minimal computational overhead, aligning with *Green AI principles* and privacy-conscious design.

---

## 🏗 System Architecture

INFAQ follows a layered architecture with a dedicated AI service component.

- *Frontend:* Flutter (cross-platform mobile application)  
- *Backend & Database:* Supabase (Authentication, PostgreSQL, Storage)  
- *AI Service:* Python (FastAPI) deployed on DigitalOcean  
- *Deployment Environment:* DigitalOcean Cloud Infrastructure  

> The AI service operates independently and exposes REST APIs that the mobile application uses for expense categorization and insight generation.

---

## 🚀 Core Features

- Secure user authentication  
- Expense creation, editing, and deletion  
- AI-based automatic category suggestion  
- Behavioral spending analysis  
- Budget limit monitoring and notifications  
- Sustainability impact detection  
- Interactive financial dashboards  
- Subscription tracking and management (e.g., Netflix, Spotify, recurring services)  
- Savings goal creation and progress tracking   

---
